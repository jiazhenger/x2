# 切换到 master 分支（只有master和develop等分支支持flutter web，stable是不行的） > flutter channel master # 开启 flutter web 支持 > flutter config --enable-web # 为已存在的项目添加 flutter web 支持，默认 web > flutter create . > flutter create web > flutter create myApp # 运行到移动设备 > flutter run # 运行到浏览器 > flutter run -d chrome # 打包成 fluuter web > flutter build web # 热加载 按 R 键 # 退出 按 Q 键
