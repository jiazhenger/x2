# 安装yarn 与 react 官方脚手架 npm install -g yarn create-react-app # 新建项目文件夹 create-react-app my-app yarn create react-app my-app # 到项目文件夹 cd my-app # 安装路由 yarn add react-router react-router-dom yarn add react-router@6 react-router-dom@6 # 启动 react yarn start # 打包 yarn run build # 将所有内建的配置暴露出来 yarn run eject
