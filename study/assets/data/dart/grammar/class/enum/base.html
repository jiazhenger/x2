/** 一、概念： 1、枚举类型是一种特殊的类，通常用来表示相同类型的一组常量。使用关键字 enum 定义枚举 2、枚举的每一个值都有一个 index 属性，index从0开始计数 3、举不能被继承，不能创建实例 */ # 定义固定数量的常量值 enum Color{ red, yellow, blue } void main( ){ int index = Color.red.index; // 枚举的索引值 => 1 Animal red = Color.red; // 获取单个枚举值值 => Animal.red List<Animal> values = Color.values; // 获取所有枚举值 => [Animal.red, Animal.yellow, Animal.blue] Animal blue = values[2]; // 获取列表值 => Animal.blue # 获取数据类型 dynamic type = red.runtimeType; // Animal dynamic type = blue.runtimeType; // Animal }
