# 安装 dart sdk https://storage.googleapis.com/dart-archive/channels/stable/release/2.7.0/sdk/dartsdk-windows-x64-release.zip <stable/dev> - 选择稳定/开发版，可选项：stable，dev <release> - 版本号，比如：2.0.0, 2.0.0-dev.64.1 <platform> - 平台，可选项：windows, macos, linux <architecture> - 系统架构，32位或64位，可选项：ia32，x64 # 配置环境变量， 在系统变量的path中配置 D:\dart-sdk\bin
