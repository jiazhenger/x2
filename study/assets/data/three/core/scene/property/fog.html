# 用法 设置雾化效果。 # 语法 scene.fog = new.Three.Fog( color, near, far ) // 场景中的物体离摄像机越远就会变得越模糊 scene.fog = new.Three.FogExp2( color, exp ) // 随着距离呈指数增长的雾化效果 # 参数 color ：必需，颜色（Color） near ：必需，雾化开始时距离，默认 1（Float） far ：必需，全雾化处距离，默认 1000（Float） exp ：必需，指数（Float） # 示例 scene.fog = newThree.Fog( 0xf7d9aa, 1, 1000 ) scene.fog = newThree.FogExp2( 0xf7d9aa, 0.02 )
