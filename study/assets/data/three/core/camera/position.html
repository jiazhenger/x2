# 用法 1、设置相机的位置后面为WEBGL的三维坐标，即：设置相机位置（眼睛位置或者说相机拍照位置） 2、在三维场景中产生漫游效果，就好像一个人走在院子等场景中，人眼看到的效果一样 3、如果不执行.lookAt( ) 方法，相机的观察方向不变，相当于生活中，摄像机移动，但是镜头指向的方向不变，或者说人一直走，但是人的头不转动 # 语法 # 拆分使用 camera.position.x = 10 camera.position.y = 10 camera.position.z = 10 # 组合使用 camera.position.set(x, y, z) # 参数 x : x 轴 y : y 轴 z : z 轴
