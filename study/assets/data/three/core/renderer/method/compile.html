# 用法 渲染器将相机放入场景编译 # 语法 renderer.compile(scene, camera) # 参数 scene ：必需，场景对象（new Three.Scene( )） camera ：必需，相机对象（new Three.Camera( )） # 示例 const scene = new Three.Scene( ) // 场景对象 const camera = new Three.OrthographicCamera( ) // 相机对象 const renderer = new Three.WebGLRenderer( ) // 渲染器 renderer.compile( scene, camera) // 编译场景与相机
