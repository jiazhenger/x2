# 用法 将纹理的所有像素复制到一个已有的从给定位置开始的纹理中 # 语法 renderer.copyTextureToTexture(position, srcTexture, dstTexture, level) # 参数 position ：必需，位置（new Three.Vector2( )） srcTexture ：必需，源纹理（new THREE.Texture( )） dstTexture ：必需，目标纹理（new THREE.Texture( )） level ：必需，层别级（Number） # 示例 const renderer = new Three.WebGLRenderer( ) renderer.copyTextureToTexture( new Three.Vector2( ), new THREE.Texture( ), 1)
