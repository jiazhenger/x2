# 用法 读取渲染目标的像素到缓冲区 # 语法 renderer.readRenderTargetPixels(renderTarget, x, y, width, height, buffer) # 参数 renderTarget ：必需，渲染目标的一个缓冲区（new Three.WebGLRenderTarget( )） x ：必需，x 轴坐标（Number） y ：必需，y 轴坐标（Number） width ：必需，宽度（Number） height ：必需，高度（Number） buffer ：必需，（Number） # 示例 const renderer = new Three.WebGLRenderer( ) const renderTarget = new Three.WebGLRenderTarget( ) renderer.readRenderTargetPixels( renderTarget, 0, 0, 100, 100, buffer)
