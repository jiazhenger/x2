# 用法 清除 # 语法 renderer.clear(color, depth, stencil) # 参数 color ：可选，如果设置, 颜色会被清除（Boolean） depth ：可选，如果设置, 深度缓存会被清除（Boolean） stencil ：可选，如果设置, 模板缓存会被清除（Boolean） # 示例 const renderer = new Three.WebGLRenderer( ) renderer.clear( true, false, false ) === renderer.clearColor( ) renderer.clear( false, true, false ) === renderer.Depth( ) renderer.clear( false, false, true ) === renderer.Stencil( )
