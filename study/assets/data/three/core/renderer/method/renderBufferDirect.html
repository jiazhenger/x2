# 用法 渲染一个几何体的缓冲组 # 语法 renderer.renderBufferDirect(camera, lights, fog, material, geometryGroup) # 参数 camera ：必需，相机对象（new Three.OrthographicCamera( )） lights ：必需，灯光组对象（Array,[new Three.AmbientLight( ),new Three.PointLight( )]） fog ：必需，雾对象（new Three.Fog( )） material ：必需，材质对象（new Three.MeshBasicMaterial( )） geometryGroup ：必需，几何体对象（new Three.BoxGeometry( )） # 示例 const camera = new Three.OrthographicCamera( ) // 相机对象 const lights = [new Three.AmbientLight( ),new Three.PointLight( )] // 灯光组对象 const fog = new Three.Fog( ) // 雾对象 const material = new Three.MeshBasicMaterial( ) // 材质对象 const geometryGroup = new Three.BoxGeometry( ) // 几何体对象 const renderer = new Three.WebGLRenderer( ) // 渲染器 renderer.render( camera, lights, fog, material, geometryGroup) // 渲染场景
