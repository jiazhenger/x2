# 用法 将输出canvas的大小调整为(width, height)并考虑设备像素比，且将视口从(0, 0)开始调整到适合大小 将updateStyle设置为false以阻止对canvas的样式做任何改变 # 语法 renderer.setSize(width, height, updateStyle) # 参数 width ：必需，宽度 height ：必需，高度 updateStyle ：可选，是否更新样式（Boolean） # 示例 const renderer = new Three.WebGLRenderer( ) renderer.setSize( window.innerWidth, window.innerHeight )
