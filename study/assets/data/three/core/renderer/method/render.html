# 用法 用相机(camera)渲染一个场景(scene) # 语法 renderer.render(scene, camera, renderTarget, forceClear) # 参数 scene ：必需，场景对象（new Three.Scene( )） camera ：必需，相机对象（new Three.OrthographicCamera( )） renderTarget ：必需，渲染目标的一个缓冲区（new Three.WebGLRenderTarget( )） forceClear ：可选，如果为true，颜色、深度及模板缓存将会在渲染之前清除（Boolean）。如果 forceClear 为 true，则即使渲染器的 autoClear 属性为 false，也会在渲染之前清除深度，模板和颜色缓冲区 # 示例 const scene = new Three.Scene( ) // 场景对象 const camera = new Three.OrthographicCamera( ) // 相机对象 const renderTarget = new Three.WebGLRenderTarget( ) // 渲染目标的一个缓冲区 const renderer = new Three.WebGLRenderer( ) // 渲染器 renderer.render( scene, camera, renderTarget, true) // 渲染场景
