# 用法 绕 axis 轴旋转 # 语法 Object3D.rotateOnAxis(axis, angle) # 参数 axis ： 必需，三维向量（ new Three.Vector3(0,0,0)）值为0或1 angle ： 必需，旋转角度(Number) # 示例 Object3D.rotateOnAxis(new Three.Vector3(1,0,0), Math.PI/4) // x 轴旋转 Object3D.rotateOnAxis(new Three.Vector3(0,1,0), Math.PI/4) // y 轴旋转 Object3D.rotateOnAxis(new Three.Vector3(0,0,1), Math.PI/4) // z 轴旋转
