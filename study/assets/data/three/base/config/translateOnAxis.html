# 用法 沿着向量表示的方向平移 # 语法 Object3D.translateOnAxis(axis, distance) # 参数 axis ： 必需，三维向量（ new Three.Vector3(0,0,0)）值为0或1 distance ： 必需，平移距离(Number) # 示例 Object3D.translateOnAxis(new Three.Vector3(1,0,0), 100) // x 轴上平移 Object3D.translateOnAxis(new Three.Vector3(1,1,0), 100) // x,y 轴上平移 Object3D.translateOnAxis(new Three.Vector3(1,1,1), 100) // x,y,z 轴上平移
