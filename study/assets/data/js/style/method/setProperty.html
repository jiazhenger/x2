# 用法 用于设置一个新的 CSS 属性，同时也可以修改 CSS 声明块中已存在的属性 # 语法 el.style.setProperty(propertyname, value, priority) # 参数 propertyname: 必需。一个字符串，表示创建或修改的属性 value : 可选。新的属性值 priority : 可选。字符串，规定是否需要设置属性的优先级 important。值： important : undefined : '' : # 示例 const style = document.styleSheets[0].rules[0].style const style = document.querySelector('#id').style style.setProperty('color', 'red')
