# 用法 比较两个节点，并返回描述它们在文档中位置的整数 # 语法 const node = el.compareDocumentPosition(node) # 参数 node: 必需。规定于当前节点作比较的节点 # 返回值 1 : 两个节点不属于同一个文档 2 : 第一节点位于第二个节点后 4 : 第一节点定位在第二节点前 8 : 第一节点位于第二节点内 16 : 第二节点位于第一节点内 32 : 两个节点是同一元素的两个属性 # 示例 const num = document.body.compareDocumentPosition( document.body.firstChild )
