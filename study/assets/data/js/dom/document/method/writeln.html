# 用法 1、与 write() 方法作用相同，但会在每个表达式后写一个换行符 2、在编写 <pre> 标签的内容时，该方法很有用 # 语法 document.writeln(exp1, exp2, exp3, ...) # 参数 exp: 必需。文本字符串、html字符串、js 代码字符串 # 示例 # 写入 html document.writeln('<b>Hello World</b>') document.writeln('<b>Hello'</b>, '<b>World</b>') # 写入 js document.writeln('<script>alert(0)</script>') document.writeln('<b>Hello World</b>','<script>alert(0)</script>') # 依次写入，不会覆盖 document.writeln('<b>Hello'</b>) document.writeln('<b>World'</b>)
