# 用法 1、把一个节点从另一个文档复制到该文档以便应用 2、节点可以是任何节点类型 3、如果 第二个值设置为 true，那么还要复制该节点的所有子孙节点 # 语法 document.importNode(node, deep) # 参数 node: 必需。另外一个文档的节点，可以是任何节点类型 deep: 可选。如果为 true，还要递归复制 importedNode 节点的所有子孙节点 # 示例 const iframe = document.querySelector('#iframe') const node = iframe.contentWindow.document.querySelector('#box') const myNode = document.importNode(node,true)
