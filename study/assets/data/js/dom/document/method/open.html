# 用法 打开一个新文档，并擦除当前文档的内容 # 语法 document.open(mimetype, replace) # 参数 mimetype: 可选。规定正在写的文档的类型。默认值是 text/html replace: 可选。当此参数设置后，可引起新文档从父文档继承历史条目。值为：replace # 在当前 html 文档写入内容 const doc = document.open('text/html','replace') // 打开当前文档流 doc.write( '<html><body>html 内容</body></html>') // 在当前文档流写入内容 doc.open( ) // 关闭当前文档流 # 在 iframe 中写入内容 const doc = document.getElementById('iframe').contentDocument // 返回 iframe 中的 document doc.open( ) // 打开当前文档流 doc.write( '<html><body>html 内容</body></html>') // 在当前文档流写入内容 doc.close( ) // 关闭当前文档流
