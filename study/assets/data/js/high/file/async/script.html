<script src='x.js' async></script> # async 1、HTML5新增属性，需要Chrome、FireFox、IE9+浏览器支持 2、规定一旦脚本可用，则会异步执行 3、仅适用于外部脚本 4、此方法不能保证脚本按顺序执行 5、将在 onload 事件之前完成 <script src='x.js' defer></script> # defer 1、延迟执行脚本，直到页面加载为止 2、如果脚本不会改变文档的内容，加上则可以加快处理文档的速度 3、兼容所有浏览器 4、所有设置了defer 属性的脚本按顺序执行 # type='module' // src 导入文件 <script src='x.js' type='module'></script> // import 导入文件 <script type='module'> import Utils from './utils.js' </script> 1、es6模块模属性 2、都是异步加载，不会造成堵塞浏览器，即等到整个页面渲染完，再执行模块脚本，等同于 defer
