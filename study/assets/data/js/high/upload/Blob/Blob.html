# 用法 Blob() 构造函数返回一个新的 Blob 对象。 blob的内容由参数数组中给出的值的串联组成 # 语法 const blob = new Blob(array, option) const blob = new Blob(array, { type:'', endings :'transparent' }) # 参数 name : 必需。是一个由ArrayBuffer, ArrayBufferView, Blob, DOMString 等对象构成的 Array ，或者其他类似对象的混合体，它将会被放进 Blob。DOMStrings会被编码为UTF-8 options : 必需。是一个可选的BlobPropertyBag字典，它可能会指定如下两个属性 type : 默认值为 ''，它代表了将会被放入到blob中的数组内容的MIME类型 endings : 默认值为 'transparent'，用于指定包含行结束符\n的字符串如何被写入 transparent : 代表行结束符会被更改为适合宿主操作系统文件系统的换行符 native : 用于指定包含行结束符\n的字符串如何被写入 # 示例 var parts = ['<a id="a"><b id="b">hey!</b></a>'] // 一个包含 DOMString 的数组 var blob = new Blob(parts, { type:'text/html' }) // 得到 blob => { type:'text/html', size:32 }
