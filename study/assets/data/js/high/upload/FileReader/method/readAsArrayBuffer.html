# 用法 1、用于启动读取指定的 Blob 或 File 内容 2、当读取操作完成时，readyState 变成 DONE（已完成），并触发 loadend 事件，同时 result 属性中将包含一个 ArrayBuffer 对象以表示所读取文件的数据 # 语法 reader.readAsArrayBuffer(blob) # 参数 blob: 必需。Blob 类型 或 File 类型 # 示例 funciton upload(e){ const file = e.target.files[0] const reader = new FileReader( ) reader.readAsArrayBuffer(file) return new Promise(resolve => { reader.onload = function( e ){ resolve(this.result) } }) }
