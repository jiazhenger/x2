# 用法 1、可以将 Blob 或者 File 对象转根据特殊的编码格式转化为内容(字符串形式) 2、这个方法是异步的，也就是说，只有当执行完成后才能够查看到结果，如果直接查看是无结果的，并返回undefined 3、也就是说必须要挂载 实例下的 onload 或 onloadend 的方法处理转化后的结果 4、当转化完成后， readyState 这个参数就会转换 为 done 即完成态， onloadend 挂载的事件会被触发，并可以通过事件返回的形参得到中的 FileReader.result 属性得到转化后的结果 # 语法 reader.readAsText(blob, encoding) # 参数 blob: 必需。Blob 类型 或 File 类型 encoding: 可选。传入一个字符串类型的编码类型，如缺省，则默认为 utf-8 # 示例 funciton upload(e){ const file = e.target.files[0] const reader = new FileReader( ) reader.readAsText(file) return new Promise(resolve => { reader.onload = function( e ){ resolve(this.result) } }) }
