# 用法 向 FormData 中添加新的属性值，FormData 对应的属性值存在也不会覆盖原值，而是新增一个值，如果属性不存在则新增一项属性值 # 语法 fd.set(name, value, filename) # 参数 name: 必需。字段名称 value: 必需。字段的值，如果是传入两个参数的方式，那么该值是一个 USVString，如果不是，将被转成一个字符串。如果是传入三个参数的方式，那么该值将是一个布尔值（Blob），文件（File），或者一个 USVString，如果不是，将被转成一个字符串 filename: 可选。传给服务端的文件名（一个USVString）。Blob 对象的默认文件名是 'blob' # 示例 const fd = new FormData( ) fd.append('name', 'javascript') set.set('name', 'flutter') // 覆盖值 set.set('age', '20') // 新增值 set.set('file', e.target.files[0], 'pic.png') // 将文件名传给服务器
