# 用法 属性包含当前正在执行的函数 # 语法 arguments.callee # 返回值 当前正在执行的函数 # 注释 1、callee 是 arguments 对象的一个属性 2、它可以用于引用该函数的函数体内当前正在执行的函数 3、这在函数的名称是未知时很有用，例如在没有名称的函数表达式 (也称为“匿名函数”)内 # 警告 在严格模式下，ES5 禁止使用 arguments.callee( )。当一个函数必须调用自身的时候, 避免使用 arguments.callee( ), 通过要么给函数表达式一个名字,要么使用一个函数声明 # 示例1 ( function( a ) { alert(a) return arguments.callee // 表示函数自身 }(0)(1)(2)) # 示例2 [1,2,3,4,5].map(function (n) { return !(n > 1) ? 1 : arguments.callee(n - 1) * n })
