# 用法 一个对应于传递给函数的参数的类数组对象 # 语法 arguments # 返回值 一个类似于数组的数据 # 注释 1、arguments 对象是所有（非箭头）函数中都可用的局部变量 2、可以使用arguments对象在函数中引用函数的参数 3、此对象包含传递给函数的每个参数，第一个参数在索引0处 4、arguments对象不是一个 Array 。它类似于Array，但除了length属性和索引元素之外没有任何Array属性 # 警告 在严格模式下，arguments对象已与过往不同。arguments[@@iterator]不再与函数的实际形参之间共享，同时caller属性也被移除 # 示例 function my( ){ var args0 = arguments[0] // 第 1 个参数 => 1 var args1 = arguments[1] // 第 2 个参数 => 2 var args2 = arguments[2] // 第 3 个参数 => 3 } my(1, 2, 3) # 将类似数组的对象 转为数组 var arg = Array.prototype.slice.call( arguments ) var arg = Array.prototype.slice.apply( arguments ) var arg = Array.prototype.slice.bind( arguments )( ) var args = [ ].slice.call( arguments ) const args = Array.from( arguments ) const args = [ ...arguments ]
