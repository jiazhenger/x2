. : 匹配单个字符，除了换行(\n)和行结束符 \w : 匹配单词字符 \W : 匹配非单词字符 \d : 匹配数字 \D : 匹配非数字字符 \s : 匹配空白字符 \S : 匹配非空白字符 \b : 匹配单词边界 \B : 匹配非单词边界 \n : 换行符 \r : 回车符 \f : 换页符 \t : 制表符 \v : 垂直制表符 \a : alert 字符 \e : escape 字符 \0 : NULL 空字符 [\b] : 回退字符 \123 : 匹配以八进制数 123 规定的字符 \x123 : 匹配以十六进制数 123 规定的字符 \u123 : 匹配以十六进制数 123 规定的 Unicode 字符
