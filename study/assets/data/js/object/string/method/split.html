# 用法 把一个字符串分割成字符串数组 # 语法 stringObject.split(separator,howmany) # 参数 separator: 必需。将被连接为一个字符串的一个或多个字符串对象 howmany: 可选。该参数可指定返回的数组的最大长度。如果设置了该参数，返回的子串不会多于这个参数指定的数组。如果没有设置该参数，整个字符串都会被分割，不考虑它的长度 # 返回值 数组 # 注释 1、如果把空字符串 ("") 用作 separator，那么 stringObject 中的每个字符之间都会被分割 2、String.split( ) 执行的操作与 Array.join( ) 执行的操作是相反的
