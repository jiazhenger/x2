# 用法 1、用于检测字符串是否以指定的子字符串开始，如果是以指定的子字符串开头返回 true，否则 false 2、对大小写敏感 # 语法 const bool = stringObject.startsWith(searchvalue, start) # 参数 searchvalue : 必需。要查找的字符串 start : 可选。查找的开始位置，默认为 0 # 示例 const str = 'abc' const bool = str.startsWith('a') // => true const bool = str.startsWith('b') // => false const bool = str.startsWith('b', 1) // => true
