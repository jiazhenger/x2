# 用法 设置小时 # 语法 (new Date).setHours( hour, min, sec, millisec ) (new Date).setUTCHours( hour, min, sec, millisec ) # 参数 hour : 必需。表示小时的数值，介于 0（午夜） ~ 23（晚上11点） 之间，以本地时间计（下同） min : 可选。表示分钟的数值，介于 0 ~ 59 之间 sec : 可选。表示秒的数值，介于 0 ~ 59 之间 millisec : 可选。表示毫秒的数值，介于 0 ~ 999之间 # 示例 const date = new Date( ) date.setHours( 15 ) date.setUTCHours( 15, 35, 1 )
