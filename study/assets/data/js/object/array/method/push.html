# 用法 向数组的末尾添加一个或多个元素，并返回新的长度 # 语法 const length = arrayObject.push(newelement1, newelement2, ..., newelementX) # 参数说明 newelement1: 必需。要添加到数组的第一个元素 newelement2: 可选。要添加到数组的第二个元素 newelement3: 可选。可添加多个元素 # 返回值 把指定的值添加到数组后的新长度 # 注释 1、按参数顺序添加到 arrayObject 的尾部 2、直接修改 arrayObject，而不是创建一个新的数组 3、使用数组提供的先进后出栈的功能
