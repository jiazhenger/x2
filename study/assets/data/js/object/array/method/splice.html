# 用法 向数组中添加或删除项目，然后返回被删除的项目 # 语法 const arr = arrayObject.splice(index, length, item1, ..., itemX) # 参数说明 index: 必需。整数，规定添加/删除项目的位置，使用负数可从数组结尾处规定位置 length: 必需。要删除的项目数量。如果设置为 0，则不会删除项目 item1, ..., itemX: 可选。可添加多个元素 # 返回值 包含被删除项目的新数组 # 注释 1、可删除从 index 处开始的零个或多个元素，并且用参数列表中声明的一个或多个值来替换那些被删除的元素 2、如果从 arrayObject 中删除了元素，则返回的是含有被删除的元素的数组
