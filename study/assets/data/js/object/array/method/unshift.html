# 用法 向数组的开头添加一个或更多元素，并返回新的长度 # 语法 const length = arrayObject.push(newelement1, newelement2, ..., newelementX) # 参数说明 newelement1: 必需。要添加到数组的第一个元素 newelement2: 可选。要添加到数组的第二个元素 newelement3: 可选。可添加若干个元素 # 返回值 arrayObject 的新长度 # 注释 1、将把它的参数插入 arrayObject 的头部，并将已经存在的元素顺次地移到较高的下标处，以便留出空间 2、该方法的第一个参数将成为数组的新元素 0，如果还有第二个参数，它将成为新的元素 1，以此类推 3、不创建新的创建，而是直接修改原有的数组
