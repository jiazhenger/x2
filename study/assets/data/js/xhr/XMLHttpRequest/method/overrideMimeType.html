# 用法 1、指定一个MIME类型用于替代服务器指定的类型，使服务端响应信息中传输的数据按照该指定 MIME 类型处理 2、此方法必须在派方法之前调用方为有效 3、XMLHttpRequest 默认 MIME 类型为 'text/xml' # 语法 xhr.overrideMimeType(mimeType) # 参数 mimeType : 必需。一个 DOMString 指定具体的 MIME 类型去代替有服务器指定的 MIME 类型. 默认 text/xml # 示例 xhr.overrideMimeType('text/plain') xhr.overrideMimeType('text/html')
