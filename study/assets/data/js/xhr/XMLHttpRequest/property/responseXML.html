# 用法 1、一个只读值，它返回一个包含请求检索的或或 xml 的Document，如果请求未成功，尚未发送，或者检索的数据无法正确解析为xml或HTML，则为NULL 2、默认是当作 'text/xml' 来解析 3、当 responseType 设置为 'Document' 并且请求已异步执行时，响应将被当作 'text/html' 来解析 4、responseXML 对于任何其他类型的数据以及 data:URL 为为空 5、如果来自服务器的响应是 XML，而且需要作为 XML 对象进行解析，请使用 responseXML 属性 6、如果服务器没有明确指出Content-Type头是 'text/xml' 还是 'application/xml'、你可以使用 XMLHttpRequest.overrideMimeType() 强制 XMLHttpRequest 解析为 xml # 语法 const text = xhr.responseXML # 示例 xhr.open('GET', /api/search, true) xhr.send( ) const text = xhr.responseXML
