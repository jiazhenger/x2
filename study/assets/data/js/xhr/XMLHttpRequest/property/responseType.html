# 用法 1、一个枚举类型的属性，返回响应数据的类型 2、它允许我们手动的设置返回数据的类型.如果我们将它设置为一个空字符串，它将使用默认的 Text 类型 # 语法 const responseType = xhr.responseType // 获取响应数据的类型 # 设置 ResponseType xhr.responseType = ' ' : 字符串。为空字符串时，采用默认类型 DOMString、与设置为 text 相同 arraybuffer : 返回包含二进制数据的 JavaScriptArrayBuffer blob : 返回包含二进制数据的Blob对象 document : 返回 HTML Document 或 XML XMLDocument、这取决于接收到的数据的MIME类型 json : 返回一个 JavaScript 对象 text : 返回一个以 DOMString 对象表示的文本 ms-stream : 返回下载流的一部分。此响应类型仅允许下载请求，仅 ie 支持
