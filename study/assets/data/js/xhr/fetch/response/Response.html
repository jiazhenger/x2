# 用法 Response( ) 构造函数创建了一个新的 Response 对象 # 语法 const headers = new Response(body, init) # 参数 body : 可选。一个定义Response中Body的对象 Blob : new Blob( ) BufferSource : FormData : new FormData( ) URLSearchParams : new URLSearchParams('a=0&b=1&c=2') USVString : 类似 DOMString init : 可选。一个参数(选项)对象，包含要应用到 Response 上的任何自定义设置 status : 回应的状态码，例如：200 statusText : 状态码关联的状态消息，例如：OK headers : 响应上的任何标头。包含了一个 new Headers( ) 对象或满足对象语法的ByteString键/值对 # 用法 const headers = new Response(new Blob(),{ status : 200 400 statusText : 'OK' 'SuperSmashingGreat' headers : new Headers( ) new Headers({ 'Content-Type', 'image/jpeg' }) { 'Content-Type', 'image/jpeg' } })
