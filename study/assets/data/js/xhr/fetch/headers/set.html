# 用法 可以在已经声明中的 headers 对象修改已有的一组键值对或者创建一个新的键值对 # 语法 headers.set(name, value) # 参数 name : 必需。要追加给标头对象的 HTTPHeader 名称 value : 必需。要追加给标头对象的 HTTPHeader 值 # 示例 headers.set('Content-Type', 'application/json;charset=utf-8') headers.set('Content-Type', 'application/x-www-form-urlencoded') headers.set('Content-Type', 'multipart/form-data') headers.set('Content-Type', 'image/jpeg') headers.set('Accept-Encoding', 'gzip') headers.set('X-Token', token)
