# 用法 在一个 Headers 对象内部，Headers 接口的 append( ) 方法可以追加一个新值到已存在的标头上，或者新增一个原本不存在的头 # 语法 headers.append(name, value) # 参数 name : 必需。要追加给标头对象的 HTTPHeader 名称 value : 必需。要追加给标头对象的 HTTPHeader 值 # 示例 headers.append('Content-Type', 'application/json;charset=utf-8') headers.append('Content-Type', 'application/x-www-form-urlencoded') headers.append('Content-Type', 'multipart/form-data') headers.append('Content-Type', 'image/jpeg') headers.append('Accept-Encoding', 'gzip') headers.append('X-Token', token)
