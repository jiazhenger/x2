# 用法 功能和 reduce() 功能是一样的，不同的是 reduceRight() 从数组的末尾向前将数组中的数组项做累加 # 语法 const num = arr.reduceRight( function(total,value,index,arr){ return value > 1 }, initialValue) # 参数 total : 必需。初始值, 或者计算结束后的返回值 value : 必需。当前元素 index : 可选。当前元素的索引值 arr : 可选。当前元素所属的数组对象 initialValue : 可选。传递给函数的初始值 # 返回值 一个布尔值 # 注释 对于空数组是不会执行回调函数的 # 常用示例 const num = arr.reduceRight((total, value)=>{ return total + value })
