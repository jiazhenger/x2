# 用法 定义一个具有指定参数的函数 # 语法 function name(param1,param2,...) { statements } # 参数说明 name: 必需，函数名 param1, param2, ...: 可选，要传递给函数的参数的名称。不同引擎中的最大参数数量不同 statements: 可选，包含函数体的语句 # 返回值 默认情况下，函数是返回 undefined 的。想要返回一个其他的值，函数必须通过一个 return 语句指定返回值 # 有条件的创建函数 函数可以被有条件来声明，这意味着，函数声明可能出现在一个 if 语句里，但是，这种声明方式在不同的浏览器里可能有不同的效果。因此，不应该在生成环境代码中使用这种声明方式，应该使用函数表达式来代替 if(true){ var fn = function( ){ } // 函数表达式 } # 函数声明提升，即：可先执行后定义 JavaScript 中的函数声明被提升到了函数定义。你可以在函数声明之前使用该函数 start( ) // 先执行 function start( ){ } // 后定义 start( ) // 后执行
