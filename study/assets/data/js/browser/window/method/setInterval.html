# 用法 1、按照指定的周期（以毫秒计）来调用函数或计算表达式 2、会不停地调用函数，直到 clearInterval() 被调用或窗口被关闭。由 setInterval() 返回的 ID 值可用作 clearInterval() 方法的参数 # 语法 window.setInterval(function, millisec) setInterval(function, millisec) # 参数 function: 必需。要调用的函数或要执行的代码串 millisec: 可选。周期性执行或调用 code 之间的时间间隔，以毫秒计 # 示例 let clear= setInterval(function( ){ }, 100) let clear= setInterval('start( )', 100) # 清除计时器 clearInterval(clear)
