# 用法 1、用于在指定的毫秒数后调用函数或计算表达式 2、setTimeout() 只执行 code 一次。如果要多次调用，请使用 setInterval() 或者让 code 自身再次调用 setTimeout() # 语法 window.setTimeout(function, millisec) setTimeout(function, millisec) # 参数 function: 必需。要调用的函数或要执行的代码串 millisec: 可选。周期性执行或调用 code 之间的时间间隔，以毫秒计 # 示例 let clear= setTimeout(function( ){ }, 100) let clear= setTimeout('start( )', 100) # 清除计时器 clearTimeout(clear)
