# 将组件或模板添加到 body 后面 <teleport to = 'body' // 目标 dom 容器 :disabled = 'false' // true：不添加到目标容器，直接显示在声明位置；false: 默认，添加到目标容器 '> <div></div> <my-component /> </teleport> # 添加组件或模板到目标容器 <teleport to='#id' '><div></div></teleport> <teleport to='.calss' '><div></div></teleport> <teleport to='html' '><my-component /></teleport> <div id='id' class='class' ></div> # 添加多个组件或模板到目标容器 <teleport to='#name' '><div></div></teleport> <teleport to='#name' '><my-component /></teleport> <div id='name' ></div>
