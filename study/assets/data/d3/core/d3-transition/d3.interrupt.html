# 用法 1、中断指定 node 上指定 name 的活跃的过渡，并取消未执行的指定 name 的过渡(如果存在的话) 2、如果没有指定 name 则使用 null。参考 selection.interrupt # 语法 d3.interrupt(node, name) # 参数 node ：必需，node 节点（node） name ：可选，默认为 null # 示例 d3.selectAll('circle') .transition( ) .delay((d, i) => i * 50 ) .on('start',function repeat( ){ d3.interrupt(this). })
