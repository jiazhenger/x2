# 用法 1、注册一个自定义补间指定的名称（name）/em> 2、当在过渡开始时，指定的工厂函数将被过渡中每个选定的元素调用，传递该元素的数据（d）和索引（ⅰ）作为参数，元素作为上下文（this） 3、工厂应反回补间函数，在整个过渡过程中被调用 4、补间函数之后被反复调用，传递当前的归一化时间t在[0，1]范围内 5、工厂应反回补间函数，在整个过渡过程中被调用 6、如果工厂返回null，则补间不会在选定元素上运行 7、 补间函数是用来内部实现attr和style补间，并可以用来对其它文档内容进行内插 # 语法 transition.tween(name, factory) # 参数 name ：必需，自定义补间名称（字符串） factory ：必需，工厂函数，反回补间函数，在整个过渡过程中被调用（常量、函数） # 函数参数 d : 当前数据 i : 当前索引 a : 当前 dom 元素数组 t : 范围 0~1 this : 当前 dom 元素 # 示例 # 终值是常量：所有的元素都被指定为相同的样式属性值 transition.tween('myName',function(d, i, a){ const f = d3.interpolateRound(0, 1000) // 内容从 0 到 1000 依次递增 return function(t) { this.textContent = f(t) this.innerText = f(t) this.innerHTML = f(t) } })
