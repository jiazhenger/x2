# 用法 1、根据指定的补间函数指定的名称过渡CSS样式属性的值 2、可选参数优先级也可以指定为 null 或者字符串 'important'（不带感叹号） 3、延迟总是相对一系列过渡中的第一个而言 # 语法 transition.style(name, tween, priority) # 参数 name ：必需，属性名（字符串） value ：必需，属性值（常量、函数） priority ：可选，优先级（null、important） # 函数 this 指向当前 dom # 示例 # value 是常量：所有的元素都被指定为相同的样式属性值 transition.style('color','red') # 优先级： transition.style('color','red',null) transition.style('color','red','important') # value 是函数：该函数为每个选定的元件（按顺序）计算属性值，被传递的参数是当前数据d与当前索引i，用this表示当前的DOM元素 transition.style('color',function(data, index){ if( i%2 === 0 ){ return 'red' // 返回值被用于设置每个元素的属性 }else{ return 'blue' // 返回值被用于设置每个元素的属性 } })
