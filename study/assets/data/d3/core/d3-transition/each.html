# 用法 1、如果指定了类型（type），就为过渡事件增加一个侦听器(listener)，同时支持 'start', 'end' 和 'interrupt' 2、侦听器会被过渡中每个单独的元素调用 # 语法 transition.each(type, listener) # 参数 name ：必需，样式名（字符串） value ：必需，样式值（常量、函数） # 函数 this 指向当前 dom # 示例 d3.transition( ).duration(750).ease(d3.easeLinear).each(function(data, index){ d3.selectAll('.foo').transition( ).style('opacity', 0).remove( ) }).transition( ).each(function(data, index){ d3.selectAll('.bar').transition( ).style('opacity', 0).remove( ) })
