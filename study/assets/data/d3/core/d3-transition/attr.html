# 用法 1、通过改变属性的值来实现过渡变化 2、过渡的初始值是当前属性值（一定要事先设定一个初始值，如果你不想坏的意外），结束值是指定的值 3、延迟总是相对一系列过渡中的第一个而言 # 语法 transition.attr(name, value) # 参数 name ：必需，样式名（字符串） value ：必需，样式值（常量、函数） # 函数 this 指向当前 dom # 示例 # value 是常量：所有的元素被给予相同的属性值 transition.attr('style','color:red') # value 是函数：该函数为每个选定的元件（按顺序）计算属性值，被传递的参数是当前数据d与当前索引i，用this表示当前的DOM元素 transition.attr('width',function(data, index){ if( i%2 === 0){ return '100px' // 返回值被用于设置每个元素的属性 }else{ return '500px' } })
