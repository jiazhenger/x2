# 用法 1、返回基于某个颜色的更暗的副本 2、如果指定了 k 则表示调整的暗度系数 3、如果没有指定 k 则默认为 1 4、这个操作的实现依赖于具体的颜色空间 # 语法 color.darker(k) # 参数 k ： 必需，亮度系数 # 示例 const color = d3.color('red') const r = color.darker( ) // => Rgb {r: 178.5, g: 0, b: 0, opacity: 1} const r = color.darker(2) // => Rgb {r: 124.94999999999999, g: 0, b: 0, opacity: 1}
