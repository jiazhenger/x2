# 用法 1、将 CSS Color Module Level 3 指定的颜色字符串转换并返回 RGB 或 HSL 颜色表示 2、如果指定的字符串不可用, 则返回null # 语法 const color = d3.color(specifier) # 参数 specifier ： 必需，颜色字符串 # 示例 const color = d3.color('red') // => Rgb {r: 255, g: 0, b: 0, opacity: 1} const color = d3.color('#cff') // => Rgb {r: 204, g: 255, b: 255, opacity: 1} const color = d3.color('rgb(10,20,25)') // => Rgb {r: 10, g: 20, b: 25, opacity: 1} const color = d3.color('rgba(0,0,0,.5)') // => Rgb {r: 0, g: 0, b: 0, opacity: 0.5}
