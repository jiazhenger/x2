# 用法 1、限定指定的命名，可能不包含命名空间前缀 2、如果name包含了冒号(:)，则冒号之前的子串被作为命名空间前缀注册到 d3.namespaces 3、返回一个包含 space 和 local 属性的对象用以描述完整的命名空间 URL 和 local 名称 # 语法 d3.namespace(name) # 参数 name ：必需，命名空间名称（node） # 示例 const r = d3.namespace('svg:text') // {space: "http://www.w3.org/2000/svg", local: "text"} const r = d3.namespace('xlink:href') // {space: "http://www.w3.org/1999/xlink", local: "href"}
