# 用法 为每个选中的元素根据指定的 type 按序派发一个 custom event. 可以指定一个可选的 parameters 映射来设置事件的附加属性 # 语法 selection.dispatch(type, parameters) # 参数 type ：必需，类型（字符串 parameters ：可选，附加属性 { bubbles: true, // 如果为 true, 这个事件会冒泡给祖先节点 cancelable : true, // 如果为 true, 则 event.preventDefault 会被允许 detail: abc, //任何关联事件的其他自定义数据 } # 示例 selection.dispatch('myEvent', { bubbles:true, cancelable :true, detail:{ } })
