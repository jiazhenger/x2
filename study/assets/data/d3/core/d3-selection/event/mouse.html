# 用法 1、返回 current event 相对于指定 container 的 x 和 y 坐标 2、container 可以是一个 HTML 或 SVG 容器元素，比如 G element 或者 SVG element 3、坐标以二元数组的形式返回: [x, y] # 语法 d3.mouse(container) # 参数 container ：必需，node（html、svg 节点）） # 示例 # 获取元素 d3.mouse(document.querySelector('#box')) // => [ 10, 20 ] # 函数添加属性 d3.on('click',function(data,index){ const xy = d3.mouse(this) // => [ 10, 20 ] })
