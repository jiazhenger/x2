# 用法 1、调用指定的 listener, 使用指定的 that 作为 this 上下文并传递指定的 arguments(如果有的话) 2、在调用期间，d3.event 被设置为指定的 event; 当事件监听器返回(或抛出错), d3.event 被重置为之前的值 3、此外，设置 event.sourceEvent 为 d3.event 的预设值，允许自定义事件对原生事件的引用 4、返回由 listener 返回的值 # 语法 d3.customEvent(event, listener, that, arguments) # 参数 event ：必需，事件名（字符串 listener ：必需，监听函数（函数、null that ：可选，指定类型 arguments ：可选，指定类型
