# 用法 1、返回指定 event 相对于指定 container 的 x 和 y 坐标 2、container 可以是一个 HTML 或 SVG 容器元素，比如 G element 或者 SVG element 3、坐标以二元数组的形式返回: [x, y] 4、如果 touches 没有被指定则默认为当前事件的 touches 属性 # 语法 d3.clientPoint(container, event) # 参数 container ：必需，node（html、svg 节点）） event ：必需，事件名称 # 示例 # 获取元素 d3.clientPoint(document.querySelector('#box'),'click') // => [x, y]
