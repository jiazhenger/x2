# 用法 1、返回 current event(当前触摸事件) 相对于指定 container 的 x 和 y 坐标 2、container 可以是一个 HTML 或 SVG 容器元素，比如 G element 或者 SVG element 3、返回的坐标以二维数组的形式返回: [[x1, y1], [x2, y2], …] 4、如果 touches 没有被指定则默认为当前事件的 touches 属性 # 语法 d3.touch(container, touches) # 参数 container ：必需，node（html、svg 节点）） touches ：可选， # 示例 # 获取元素 d3.touches(document.querySelector('#box')) // => [ [x1,y1], [x2,y2] ]
