# 用法 1、返回与 current event 相关的指定标识符的触摸事件相对于指定 container 的 x 和 y 坐标 2、container 可以是一个 HTML 或 SVG 容器元素，比如 G element 或者 SVG element 3、坐标以二元数组的形式返回: [x, y] 4、如果当前 touches 中没有与指定标识符相关联的事件则返回 null 5、当仅仅有部分触摸移动的时候忽略 touchmove 会很有用 6、如果没有指定 touches 则默认为当前事件的 changedTouches 属性 # 语法 d3.touches(container, touch, identifier) # 参数 container ：必需，node（html、svg 节点）） touches ：可选， identifier ：可选，标识符 # 示例 # 获取元素 d3.touch(document.querySelector('#box')) // => [ [x,y] ]
