# 用法 1、如果 data 数据与节点不一致，且数组元素个数比节点多，就插入节点补足并相应更新文本 2、比 data 元素少出来的节点是 enter( ) # update、enter、exit 原理 update ：节点数量 == 数据数量的部分 enter ：节点数量 < 数据数量的部分 exit ：节点数量 > 数据数量的部分 # 语法 selection.enter( ) # 拆分示例 # 选择 #box 中的 p 元素 const p = d3.select('#box').selectAll('p') # 获取 update 部分 const update = p.data([1,2,3]) # 获取 enter 部分 const enter = update.enter( ) # update 部分处理：更新属性值 update.text( d => d) # enter 部分处理：添加元素后赋予属性值 enter.append('p').text( d => d) # 组合示例 d3.select('#box').selectAll('p').data([1,2,3]).text( d => d).enter( ).append('p').text( d => d)
