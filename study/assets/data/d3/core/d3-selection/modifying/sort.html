# 用法 返回一个新选择集，其中包含了当前选择集中所有元素的经过 compare 函数排序之后的副本。在排序后，将排序后的元素重新插入原来的文档中替换未排序的元素/em> # 语法 selection.sort(compare) # 参数 compare ：可选，比较函数，默认 d3.ascending( ) # 示例 # 升序 d3.select('#box').selectAll('p').data([1,2,3]).sort( ) # 自定义：升序 d3.select('#box').selectAll('p').data([1,2,3]).sort(function(a, b){ return a - b }) # 自定义：降序 d3.select('#box').selectAll('p').data([1,2,3]).sort(function(a, b){ return b - a })
