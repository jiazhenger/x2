# 用法 1、返回一个将当前选择集和指定的 other 选择集合并之后的新的选择集 2、返回的选择集与当前选择集有相同数量的分组以及相同的父元素 3、当前选择集中的任何空缺 (null) 都会被指定的 other 中的对应的元素填充(如果存在的话)，如果 other 选择集有附加的组或者父元素，则将其忽略 4、这个方法通常用来在 data-join 操作之后合并 enter 和 update。在单独修改输入和更新元素之后对其进行合并执行其他的操作而不需要额外的代码 # 语法 selection.merge(other) # 参数 other ：必需，要合并的 selection # 函数 this 指向当前 dom # 示例 var circle = svg.selectAll('circle').data(data) // UPDATE .style('fill', 'blue') circle.exit( ).remove( ) // EXIT circle = circle.enter( ).append('circle') // ENTER .style('fill', 'green') .merge(circle) // ENTER + UPDATE .style('stroke', 'black');
