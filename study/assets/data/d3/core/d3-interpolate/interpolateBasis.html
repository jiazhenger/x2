# 用法 根据给定的一组 values 返回一个 B-splibe 插值器, 输入值必须为数值类型，隐式控制点的生成会使得在 t = 0 时返回 values[0] 并且 t = 1 时返回 values[values.length - 1], 参考 d3.curveBasis # 语法 const interpolate = d3.interpolateBasis(values/s>)(t) # 参数 values ：必需，数组 t ：区间 [ 0, 1 ] # 示例： const r = d3.interpolateBasis([ 0, 10 ])(.5) // => 5 const r = d3.interpolateBasis([ 10, 0 ])(.5) // => 5
