# 用法 1、根据指定的 interpolator 返回 n 个等间隔的均匀采样, 其中 n 是一个大于 1 的整数。第一个采样点总是取 t = 0 时的值而最后一个值总是取 t = 1 处的值。这个方法可以从给定的插值器中取固定数量的等间隔的值 2、这种方法不适用于不返回副本的插值器, 比如 d3.interpolateArray, d3.interpolateDate 和 d3.interpolateObject。对于这些插值器，你必须对其包装，每次插值时返回一个副本 # 语法 const interpolate = d3.quantize(interpolate, n) # 参数 interpolate ：必需，一个插值器 n ：必需，数组 # 示例： const r = d3.quantize(d3.interpolateRgb.gamma(2.2), 5) // => [ƒ, ƒ, ƒ, ƒ, ƒ]
