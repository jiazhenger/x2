# 用法 1、在两个颜色 a 和 b 之间创建一个 Lab 颜色空间的插值器 2、a 和 b 不一定使用 Lab 表示，它们将会适用 d3.lab 转为 Lab 表示。返回的值使用 RGB 字符串表示 # 语法 const interpolate = d3.interpolateLab(a, b)(t) # 参数 a ：必需，开始值 b ：em>必需，结束值 t ：区间 [ 0, 1 ] # 示例： const r = d3.interpolateLab('#f00', '#cff')(.5) // => rgb(253, 162, 128) const r = d3.interpolateLab('#cff', '#f00')(.5) // => rgb(253, 162, 128)
