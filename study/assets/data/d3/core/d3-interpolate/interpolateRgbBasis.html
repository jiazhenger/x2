# 用法 1、根据指定的颜色数组返回一个 uniform nonrational B-样条插值器, 这些颜色都会被转为 RGB color space 2、隐式控制点的生成会使得 t = 0 时返回 colors[0] 并且在 t = 1 时返回 [colors.length - 1] 3、目前支持不透明度的插值 # 语法 const interpolate = d3.interpolateRgbBasis(colors)(t) # 参数 colors ：必需，颜色数组 # 示例： const r = d3.interpolateRgbBasis(['#f00', '#cff'])(.5) // => rgb(230, 128, 128)
