# 用法 1、根据指定的颜色数组返回一个 uniform nonrational B-样条插值器, 这些颜色都会被转为 RGB color space 2、隐式的控制点是重复的，这样在 t 处于 [0, 1] 时返回的结果是循环重复的 3、创建一个循环颜色比例尺时是有用的 4、目前支持不透明度的插值 # 语法 const interpolate = d3.interpolateRgbBasisClosed(colors)(t) # 参数 colors ：必需，颜色数组 # 示例： const r = d3.interpolateRgbBasisClosed(['#f00', '#cff'])(.5) // => rgb(221, 170, 170)
