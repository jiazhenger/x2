# 用法 1、使用指定的参数 comparator 来给当前键值排序，等效于 d3.descending。如果没有指定 comparator 参数键的排序则返回 undefined 2、注意：只影响实体操作符的结果；map操作符返回键的顺序永远是undefined，无论什么比较器 # 语法 nest.sortKeys(comparator) # 参数 comparator ：必需，函数 # 示例 const arr = d3.nest( ).key(function(value){ return value.year }).sortKeys(d3.ascending/d3.descending).entries([{ year:2020, name:'d3' },{ year:2020, name:'three' }])
