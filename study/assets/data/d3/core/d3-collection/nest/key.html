# 用法 注册一个新的键函数function。键函数将被输入数组中的每个元素调用，并且必须返回一个用于分配元素到它的组内的字符串标识符 # 语法 nest.key(function) # 参数 function ：必需，函数 # 示例 const arr = d3.nest( ).key(function(value){ return value.year }).entries([{ year:2020, name:'d3' },{ year:2020, name:'three' }]) // => [{key:"2020", values:[{ year:2020, name:'d3' },{ year:2020, name:'three' }]}]
