# 用法 1、若 map 有指定 key 的实体，删除此实体并返回 true。否则，此方法不做任何操作，返回false 2、删除整个 Map 数据，而不是删除 Map 中的某个 key 3、若 map 有指定 key 的实体，删除此实体并返回 true。否则，此方法不做任何操作，返回false # 语法 map.remove(key) # 参数 key ：可选，Map 的 key 值 # 示例 const m = d3.map({ a:0, b: 1}) // => Map {$a:0, $b;1} const bool = m.remove('a') // => true const r = m // => Map {$b;1}
