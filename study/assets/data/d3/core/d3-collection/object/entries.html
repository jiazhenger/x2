# 用法 1、返回一个包含对象(object)(一个关联数组)中名称以及值(键和值， key and value)的数组 (array) 2、每一个实体都是有键值对的对象 3、返回数组的顺序未定义 # 语法 d3.entries(object) # 参数 object ：必需，一个数组或对象类型数据 # 示例 # 对象 const arr = d3.entries({ a:0, b: 1}) // => [{key:'a', value:0},{key:'b', value:1}] # 数组 const arr = d3.entries([ 'a', 'b' ]) // => [{key:'0', value:'a'},{key:'1', value:'b'}] # 字符串 const arr = d3.entries('ab') // => [{key:'0', value:'a'},{key:'1', value:'b'}] # 数字 const arr = d3.entries(123) // => [ ]
