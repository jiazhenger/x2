# 用法 1、返回一个包含指定对象(关联数组) 属性值的数组。返回数组的顺序未定义 2、返回数组的顺序未定义 # 语法 d3.values(object) # 参数 object ：必需，一个数组或对象类型数据 # 示例 # 对象 const arr = d3.values({ a:0, b: 1}) // => ["0", "1"] # 数组 const arr = d3.values([ 'a', 'b' ]) // => ["a", "b"] # 字符串 const arr = d3.values('abc') // => ["a", "b", "c"] # 数字 const arr = d3.values(123) // => [ ]
