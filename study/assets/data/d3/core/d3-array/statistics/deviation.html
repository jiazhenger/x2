# 用法 1、返回给定数组（array）的标准差，即方差的平方根 2、如果数组的长度小于 2，返回undefined 3、此方法忽略无效值(如 NaN 和undefined) # 语法 d3.median(deviation, accessor) # 参数 array ： 必需，数组 accessor ： 可选，一个处理数据的函数 # 示例 # 无 accessor 函数 const r = d3.deviation([ ]) // => undefined const r = d3.deviation([ 1, 7, 10 ]) // => 4.58257569495584 # 有 accessor 函数，相当于 map( )，返回处理后的数组，再取方差 const r = d3.deviation([ 1, 7, 10 ],function(value, index, arr){ // => 4.58257569495584 return value + 10 })
