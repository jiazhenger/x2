# 用法 1、返回给定数组（array）中自然排序最小的值 2、如果数组为空，返回undefined。如果指定了accessor 参数，等同与在计算最小值之前调用了array.map(accessor)方法 3、不同于内置的Math.min，这个方法会忽略未定义的值 # 语法 d3.min(array, accessor) # 参数 array ： 必需，数组 accessor ： 可选，一个处理数据的函数 # 示例 # 无 accessor 函数 const min = d3.min([ 1, 5, 10 ]) // => 1 # 有 accessor 函数，相当于 map( )，返回处理后的数组，再取最小值 const min = d3.min([ 1, 5, 10 ],function(value, index, arr){ // => 11 return value + 10 })
