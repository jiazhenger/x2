# 用法 1、返回给定数组（array）的和 2、如果数组为空，返回 0 3、此方法忽略无效值(如 NaN 和 undefined) # 语法 d3.sum(array, accessor) # 参数 array ： 必需，数组 accessor ： 可选，一个处理数据的函数 # 示例 # 无 accessor 函数 const r = d3.sum([ ]) // => 0 const r = d3.sum([ 1, 5, 10 ]) // => 16 # 有 accessor 函数，相当于 map( )，返回处理后的数组，再取和 const r = d3.sum([ 1, 5, 10 ],function(value, index, arr){ // => 46 return value + 10 })
