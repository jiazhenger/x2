# 用法 1、如果指定了 domain 则设置当前直方图生成器的输入区间为指定的 domain 并返回直方图生成器 2、如果没有指定 domain 则返回当前的输入区间, 默认为 extent 3、直方图的输入区间由数组 [min, max] 定义. min 表示直方图生成器可以读取的最小值, max 表示直方图生成器可以读取到的最大值 4、任何不属于 [min, max] 的值将会被忽略 # 语法 histogram.domain(domain) # 参数 domain ：可选，一个值 # 示例 # 回当前的输入区间, 默认为 extent const histogram = d3.histogram( ).domain(x.domain( ))
