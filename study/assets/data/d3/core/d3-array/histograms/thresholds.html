# 用法 1、如果指定了 thresholds 则将 threshold generator(阈值生成器) 设置为指定的函数或数组并返回当前的直方图生成器 2、如果没有指定 thresholds 则返回当前的阈值生成器, 默认的阈值生成器参考 Sturges’ formula 实现(因此, 默认情况下, 直方图的输入数据必须为数值) 3、阈值由一组数组定义: [x0, x1, …]. 小于 x0 的值放在第一个分箱中, 小于 x1 并且大于等于 x0 的放在第二个分箱中 4、任何在 domain 之外的值将会被忽略. 第一个分箱的 bin.x0 属性总是等于最小的 domain 值, 最后一个分箱的 bin.x1 属性总是等于 domain 的最大值 5、如果指定了 count 则表示使用 count 代替 thresholds 数组, domain 将会被设置成近似的计数区间, 参考 ticks # 语法 histogram.thresholds(count) histogram.thresholds(thresholds) # 参数 count ：可选 thresholds ：可选 # 示例 const histogram = d3.histogram( ).domain(x.domain( )).thresholds(x.ticks(20))
