# 用法 使用指定参数 accessor 或者 comparator 函数返回一个二等分线。返回的对象有 left 和 right 属性，分别类似于 bisectLeft 和 bisectRight 方法 # 语法 const bisector = d3.bisector(accessor) const bisector = d3.bisector(comparator) # 参数 accessor ：必需，访问器 comparator ：必需，比较器 # 示例 # accessor const r = d3.bisector(function(d){ return d.x }).right # comparator const r = d3.bisector(function(a,b){ return a.x - b.x }).right
