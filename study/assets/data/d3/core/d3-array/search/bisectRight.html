# 用法 1、定位数组 array 中的 x 的插入点，以保持已有序列 2、参数 lo 和 hi 用来指定数组的子集 3、默认情况下整个数组都被使用 4、如果 x 在 array 中已存在，插入点在所有元素之前（右侧） 5、返回值适合用作拼接（splice）已经排序的数组array 的第一个参数 6、返回的插入点i把array 分为两个区：数组中所有array.slice(lo, i)中v < x 的v在左边，数组中所有 array.slice(i, hi)中v >= x 的v在右边 # 语法 d3.bisectRight(arry, x, lo, hi) # 参数 array ： 必需，数组 x ： 可选， lo ： 可选， hi ： 可选， # 示例 const r = d3.bisectRight([ ]) // => 0 const r = d3.bisectRight([ 1, 7, 10 ]) // => 3 const r = d3.bisectRight([ 1, 7, 10 ], 7) // => 2 const r = d3.bisectRight([ 1, 7, 10 ], 7), 10) // => 10 const r = d3.bisectRight([ 1, 7, 10 ], 7), 10, 80) // => 80
