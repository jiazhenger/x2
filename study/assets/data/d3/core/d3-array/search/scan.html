# 用法 对指定的数组执行线性扫描, 根据指定的比较器返回最小元素的索引. 如果给定的数组中没有可比较的元素 ( 比如比较时返回 NaN ) 则返回 undefined. 如果没有指定 comparator 则默认为 ascending(升序) # 语法 d3.scan(arry, comparator) # 参数 array ： 必需，数组 comparator ： 可选，对比函数 # 示例 # 默认对比 const r = d3.scan([ ]) // => undefined const r = d3.scan([ 1, 2 ]) // => 0 const r = d3.scan([ 1, 2 ]) // => 1 const r = d3.scan([ 2, 2]) // => 0 # 自定义 const r = d3.scan([ 2, 1], function(a/s>, b){ // => 1 return a - b })
