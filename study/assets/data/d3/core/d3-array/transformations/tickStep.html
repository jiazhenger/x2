# 用法 返回刻度之间的差值, 差值将相同的参数传递给 d3.ticks 时相邻的两个刻度之间的差值: 会计算一个 10 的 1, 2 或 5 的次幂 的舍入值. 要注意的是因为 IEEE 754 浮点数的存储原因, 返回的值可能不精确. 使用 d3-format 可以将其转换为人类友好的值 # 语法 d3.tickStep(start, stop, count) # 参数 start ：可选，开始索引 stop ：可选，结束索引 count ：可选，次数 # 示例 const arr = d3.tickStep(0, 10, 3) // => 5 const arr = d3.tickStep(0, 10, 4) // => 2 const arr = d3.tickStep(0, 10, 5) // => 2
